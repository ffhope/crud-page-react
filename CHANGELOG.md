# 变更日志

## [0.1.5] - 2024-12-19

### 改进
- 调整操作按钮顺序："其它操作"按钮现在放在最后
- 优化布局：基础操作 → 复制 → 其它操作，逻辑更清晰

## [0.1.4] - 2024-12-19

### 改进
- 复制JSON功能独立为单独按钮，提升用户体验
- 仅在有自定义操作时才显示"其它操作"下拉菜单
- 优化操作列布局，功能分组更清晰

## [0.1.3] - 2024-12-19

### 新增
- 自定义操作现在会自动折叠到"其它操作"下拉菜单中
- 支持危险操作的红色标识（danger: true）
- 智能显示：仅在有自定义操作时才显示下拉菜单

### 改进
- 操作列宽度固定为200px，提供更好的布局一致性
- 基础操作（查看、编辑、删除）仍显示为独立按钮
- 复制JSON功能独立为单独按钮，不占用下拉菜单空间
- 更清晰的功能分组和用户体验

### 技术细节
- 新增 `Dropdown` 和 `MoreOutlined` 组件的使用
- 优化操作列的渲染逻辑，分离基础操作和自定义操作
- 保持向后兼容性，现有配置无需修改

## [0.1.2] - 2024-12-18

### 初始发布
- 基于 React + Ant Design 的动态 CRUD 组件库
- 支持通过 JSON Schema 配置生成完整的增删改查界面
- 支持自定义 API 请求函数
- 完整的 TypeScript 类型定义
