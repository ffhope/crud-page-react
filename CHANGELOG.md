# 变更日志

## [0.1.3] - 2024-12-19

### 新增
- 自定义操作现在会自动折叠到"其它操作"下拉菜单中
- 下拉菜单包含所有自定义操作和复制JSON功能
- 支持危险操作的红色标识（danger: true）

### 改进
- 操作列宽度固定为200px，提供更好的布局一致性
- 基础操作（查看、编辑、删除）仍显示为独立按钮
- 下拉菜单自动分组，用分割线分隔不同类型的操作

### 技术细节
- 新增 `Dropdown` 和 `MoreOutlined` 组件的使用
- 优化操作列的渲染逻辑，分离基础操作和自定义操作
- 保持向后兼容性，现有配置无需修改

## [0.1.2] - 2024-12-18

### 初始发布
- 基于 React + Ant Design 的动态 CRUD 组件库
- 支持通过 JSON Schema 配置生成完整的增删改查界面
- 支持自定义 API 请求函数
- 完整的 TypeScript 类型定义
